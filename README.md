# I am i

A Chrome extension to detect the URL which contains cyrillic `Ñ–`.

## Install

`$ make setup`

## Development
`$ make`

## Build
`$ make prod`

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
